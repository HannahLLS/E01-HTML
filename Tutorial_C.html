  • Exemplo:

    casa Valor1:
    Lista de instruções;

    Lista de instruções;

    casa Valores... :

Estrutura While

  • Executa a repetição de um bloco de instruções enquanto uma condição é verdadeira. A estrutura Enquanto … Faça equivale a estrutura while em linguagem C. Para que seja possível fazer o teste, a variável de controle deve ter sido inicializada previamente. Observe que o teste da condição ocorre no início do laço. Enquanto a condição permanecer verdadeira, são executadas as instruções. Quando a condição se tornar falsa, o processamento será desviado para fora do laço.
  • Exemplo:
    #include
    int main(void)
    { int contador = 1; //declarando e inicializando a variável de controle
    while (contador <= 10) // Testando a condição
    { printf("%d ", contador); //Executando um comando dentro do laço
    contador++; //atualizando a variável de controle }
    return 0;
    }

Loop For

  • Para implemnetar o for basta especificar o nome da variável que serve de contador (e, eventualmente, o seu valor inicial, a condição sobre a variável para a qual o loop para (basicamente uma condição que testa se o valor do contador excede um limite) e, finalmente, uma instrução que incrementa (ou decrementa) o contador.
  • Exemplo:
    for (i=1; i<6; i++) {
    printf("%d", i); }
